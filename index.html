Популярные услуги

УЗИ органов малого таза

Ультразвуковое исследование для диагностики состояния матки, яичников, мочевого пузыря. Безопасно, безболезненно, информативно. Подходит для всех возрастов.

Ведение беременности

Комплексное наблюдение за беременностью: осмотры, анализы, УЗИ, подготовка к родам. Подходит для всех триместров. Индивидуальный подход к каждой пациентке.

Приём по ОМС

Бесплатный приём для женщин с полисом ОМС. Включает осмотр, консультацию, назначение анализов. Без очередей и лишних хлопот.

Гормональный профиль

Комплексный анализ гормонов для диагностики нарушений менструального цикла, бесплодия, климакса. Подготовка к ЭКО и планированию беременности.

Контрацепция

Подбор метода контрацепции, консультация по гормональным препаратам, установка ВМС. Безопасно и эффективно.
